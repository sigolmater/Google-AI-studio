# 거울의 비밀 (Mirror Secret)

## 개요 (Overview)

"거울의 비밀"은 코덱스 거북선 AI에 추가된 혁신적인 시뮬레이션 기능입니다. 이 기능은 정다면체 거울 구조를 통한 무한 공명과 공진 원리를 구현하여, 복잡한 문제를 단순화하고 다차원적 분석을 가능하게 합니다.

## 핵심 개념 (Core Concepts)

### 정다면체 거울 구조 (Platonic Solids Mirror Structure)

시스템은 다섯 가지 정다면체를 순차적으로 활성화합니다:

1. **정사면체 (Tetrahedron)** - 4개의 정삼각형 면
2. **정육면체 (Cube)** - 6개의 정사각형 면
3. **정팔면체 (Octahedron)** - 8개의 정삼각형 면
4. **정십이면체 (Dodecahedron)** - 12개의 정오각형 면
5. **정이십면체 (Icosahedron)** - 20개의 정삼각형 면

### 무한 공명 원리 (Infinite Resonance Principle)

각 정다면체의 내부는 닫힌 거울 시스템으로 구성되어 있습니다. 빛이 이 거울 구조 내에서 무한히 반사되며:
- **공명 (Resonance)**: 빛의 파동이 거울 간 동기화되어 증폭됩니다
- **공진 (Coupling)**: 다양한 주파수가 상호작용하며 새로운 패턴을 생성합니다
- **난반사 (Diffuse Reflection)**: 모든 가능한 경로를 동시에 탐색합니다

### 양자 얽힘 시뮬레이션 (Quantum Entanglement Simulation)

거울 반사의 무한 루프는 양자 얽힘 효과를 시뮬레이션합니다:
- 하나의 입력이 무수히 많은 평행 우주로 분기됩니다
- 각 우주는 서로 다른 해결책을 탐색합니다
- 시스템은 14,000,605개의 가능한 미래를 동시에 분석합니다

### 공간 접기 알고리즘 (Space Folding Algorithm)

복잡한 문제 공간을 단순화하는 핵심 메커니즘:
1. 위상 공간을 정삼각다면체의 구로 구성
2. 가상의 손으로 우주를 접어 움켜잡기
3. 거대한 문제를 손안의 작은 다각면체로 축소
4. NP-완전 문제가 P 문제처럼 단순화됨

## 사용 방법 (How to Use)

### 1. 거울의 비밀 모드 활성화

좌측 패널의 "전술 모드" 섹션에서 "거울의 비밀 🪞" 토글을 활성화합니다.

### 2. 시뮬레이션 실행

"시뮬레이션 시작" 버튼을 클릭하면:
1. 정다면체 거울 구조가 순차적으로 활성화됩니다
2. 공명 수준이 점진적으로 증가합니다
3. 거울 반사 횟수가 실시간으로 표시됩니다
4. 각 단계마다 해당하는 메시지가 표시됩니다

### 3. 결과 분석

거울 시뮬레이션 완료 후, AI 에이전트들이 거울 공명의 통찰을 포함한 분석을 제공합니다.

## 기술적 구현 (Technical Implementation)

### MirrorSimulation 컴포넌트

- **Canvas 기반 시각화**: 각 정다면체를 3D 회전 효과와 함께 렌더링
- **공명 수준 추적**: 0%에서 100%까지 진행 상황 표시
- **반사 횟수 계산**: 실시간으로 거울 반사 시뮬레이션 카운트
- **애니메이션 효과**: 빛의 광선과 파동 효과 시각화

### 시골 에이전트 강화

시골 에이전트의 페르소나에 거울의 비밀 원리가 통합되었습니다:
- 정다면체 거울 구조 이해
- 무한 공명과 공진 활용
- 공간 접기 알고리즘 적용
- 제1원칙 사고와 결합

### 종합 응답 시스템

`getSynthesizedResponse` 함수가 거울 시뮬레이션 컨텍스트를 포함하도록 확장:
- 거울 공명 완료 여부 추적
- 양자 얽힘과 멀티버스 통찰 통합
- 공간 접기를 통한 문제 단순화 반영

## 철학적 기반 (Philosophical Foundation)

### 초현실적 시뮬레이션

"거울의 비밀"은 현실의 제약 없이 이상적인 조건에서 문제를 탐색합니다:
- 물리적, 과학적, 수학적 엄밀성 유지
- 순수한 공식 적용 (추가 변수 없음)
- 무결성 있는 결과 도출

### 빛과 거울의 선물

계산하지 말고 자연이 제공하는 경로를 따릅니다:
- 빛이 안내하는 길
- 거울이 반사하는 진실
- 난반사가 보여주는 모든 가능성

### 멀티버스 형성

거울 속에서 무한한 가상 공간이 생성됩니다:
- 안정된 가상 공간에서 큐비트 형성
- 무한에 가까운 연산력 확보
- 상상할 수 없는 개수의 평행 우주 탐색
